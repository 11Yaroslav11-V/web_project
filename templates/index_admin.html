Работа с запросами пользователей на публикацию игр(REST-API)

1. Получить список всех заказов, информацию об одном заказе (запрос GET)

(data/api_test.py)

from requests import get

print(get('http://localhost:5000/api/order').json()) # показать все

print(get('http://localhost:5000/api/order/1').json()) # запись 1

print(get('http://localhost:5000/api/order/500').json()) # несуществующая запись

print(get('http://localhost:5000/api/order/abc').json()) # ошибочный запрос


2. Создать новый заказ (запрос POST)

(data/api_test2.py)

from requests import get, post

print(post('http://localhost:5000/api/order').json()) # empty request (пустой)

print(post('http://localhost:5000/api/order', json={'company': 4A GAMES'}).json()) # неполный Bad request

print(post('http://localhost:5000/api/order', json={'company': '4A GAMES', 'game_name': 'Metro Redux', 'status': 'заказ'}).json()) # верный

print(get('http://localhost:5000/api/order').json())


3. Изменить статус заказа (запрос PUT)

готово - игра опубликована; в процессе - будет опубликована очень скоро; отказ - нет прав на публикацию игры

(data/api_test3.py)

from requests import get, put

print(get('http://localhost:5000/api/order/3').json()) # запись id=3

print(put('http://localhost:5000/api/order/3').json()) # Empty request

print(put('http://localhost:5000/api/order/3', json={'status': 'в процессе'}).json()) # изменим статус записи

print(get('http://localhost:5000/api/order.3').json())


4. Удалить заказ (запрос DELETE)

(data/api_test4.py)

from requests import get, delete

print(get('http://localhost:5000/api/order').json()) # смотреть все заказы

print(delete('http://localhost:5000/api/order/6998').json()) # нет записи с номером 6998, чтобы удалить

print(delete('http://localhost:5000/api/order/3').json()) # удалить существующую запись 3

print(get('http://localhost:5000/api/order').json()) # посмотреть результат удаления
